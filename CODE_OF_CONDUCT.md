# Code of Conduct / 行为准则

We expect respectful, constructive collaboration. Harassment, abusive language, credential leakage, or unsafe data handling is not acceptable.

我们期望尊重、建设性的协作。骚扰、辱骂、泄露凭证或不安全的数据处理都不可接受。

Report concerns through repository maintainers. Maintainers may remove harmful content or block abusive contributors.

如有问题请向维护者报告。维护者可以移除有害内容或限制滥用者参与。
