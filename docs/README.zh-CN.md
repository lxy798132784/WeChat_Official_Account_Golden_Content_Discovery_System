# 全网黄金内容雷达

全网黄金内容雷达是一个本地优先的 Qt 6 桌面软件，用来接收、存储、评分和导出微信公众号文章指标。它适合在用户自己控制、合法授权的本地环境中使用：软件本身不保存账号密码，不内置 Cookie，不上传私有数据，也不提供远程爬虫服务。

## 当前边界

桌面主程序、插件运行时、本地桥、SQLite、评分、筛选、种子池、导出、测试、打包和发布流程都已经具备。真正上线采集时，还需要大王准备合法的数据来源或本地适配器，把微信文章指标转换成本文档约定的本地 JSON 载荷，发送到 `127.0.0.1` 的桥端口。
