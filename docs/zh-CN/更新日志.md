# 更新日志

## v1.0.1 - 2026-06-05

- 增加持久化运行配置，支持数据库路径、插件目录、桥端口、ADB 开关和示例数据开关。
- 增加本地桥冒烟测试按钮和命令行 `--bridge-smoke`。
- 强化生产落地文档，明确真实数据接入需要合法本地适配器。
