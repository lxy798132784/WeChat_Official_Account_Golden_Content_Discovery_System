# 快速开始：接手机、输关键词、一键操作

## 这个页面干什么

快速开始页把原来分散在“手机诊断、关键词发现、自动采集、生产中心”的步骤串成一个按钮。普通用户只需要接好安卓手机、输入关键词，然后点击“一键发现并打开文章”。

## 使用步骤

1. 手机打开开发者选项和 USB 调试。
2. 用数据线连接电脑，手机弹窗选择“允许 USB 调试”。
3. 打开软件左侧“快速开始”。
4. 在关键词框输入关键词，每行一个。
5. 高级筛选可以不改，第一次建议保持默认。
6. 点击“一键发现并打开文章”。
7. 看页面里的四个状态：手机就绪、关键词发现、打开队列、指标回传和推荐。

## 软件自动做什么

点击按钮后软件会自动完成：

- 检查 ADB 是否可用。
- 检查手机是否已授权。
- 加载本地插件和本地桥。
- 搜索关键词候选文章。
- 按阅读、点赞、评论、爆款分筛选。
- 把候选微信文章链接加入队列。
- 通过 ADB 在手机上逐篇打开文章。
- 等待合法代理适配器或回放数据把阅读、点赞、评论等脱敏指标送回本地桥。
- 刷新文章库、数据质量、趋势追踪、评分和报告入口。

## 重要边界

快速开始不是破解微信，也不会要求 Cookie、请求头、Token、证书或账号密码。

如果没有配置合法代理适配器或导入脱敏数据，软件仍然可以自动完成：

- 关键词候选发现
- 链接筛选入队
- 手机打开文章

但真实阅读数、点赞数、评论数需要来自用户自己合法可见的数据来源。

## 常见问题

### 提示手机预检失败

检查：

- 是否安装 Android Platform Tools / adb。
- 手机是否打开 USB 调试。
- 手机是否弹窗授权。
- 数据线是否支持数据传输。
- 多台手机同时连接时是否选择了目标手机。

### 没有候选链接入队

可能是筛选阈值过高。降低最低阅读、最低点赞、最低评论或最低爆款分后重试。

### 已打开文章但没有指标

说明手机打开链路成功，但指标没有回传。需要启动合法本地代理适配器，或在回放中心导入脱敏 JSON/JSONL 数据。
