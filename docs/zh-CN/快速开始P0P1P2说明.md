# 快速开始 P0/P1/P2 操作说明

## P0 主路线：电脑端找候选 + 手机打开文章

默认推荐路线仍然是：

1. 输入市场关键词。
2. 连接已授权的安卓手机。
3. 点击“一键发现并打开文章”。
4. 软件检查手机调试和授权状态，搜索外部微信文章候选页，筛选并去重候选，把微信文章链接加入队列，再通过安卓意图在手机上打开文章。

这条路线最稳定，因为它不依赖微信按钮位置、手机分辨率、输入法或微信界面变化。

## P1 补充候选来源

如果在线搜索源返回候选太少，可以在“补充来源”里粘贴下面任意一种内容，并勾选“使用补充数据或粘贴链接”：

- 脱敏结果数据数组，或包含 `articles/items/results/data` 的对象。
- 每行一个微信文章链接。

软件会把补充数据和在线搜索结果合并、按链接去重，并继续使用同一套爆款筛选条件。普通链接会直接加入手机打开队列。

## P2 高级微信内搜索

高级微信内搜索默认关闭。它是兜底模式，不是主路线。

打开后，软件可以执行一组保守的手机调试命令：

1. 启动微信。
2. 导出当前界面结构到 `/sdcard/window.xml`。
3. 点击用户填写的搜索框坐标。
4. 输入关键词。
5. 发送回车键。
6. 可选点击搜索结果坐标。

这个模式需要手动坐标，因为微信界面会受版本、设备、分辨率、输入法、登录状态和弹窗影响。即使启用，高稳定性的电脑端搜索路线仍会继续作为兜底。

## 隐私和安全边界

这个功能不会采集登录凭证、请求头、访问令牌、证书、聊天内容或账号密码。真实阅读、点赞、评论指标仍需要来自合法本地适配器或脱敏回放/导入数据。
