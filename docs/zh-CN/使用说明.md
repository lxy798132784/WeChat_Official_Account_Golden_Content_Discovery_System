# 使用说明

## 软件用途

全网黄金内容雷达用于发现微信公众号中的高价值文章。它不是远程爬虫，也不是账号托管工具，而是一个本地分析工作台：本地适配器把合法获取到的文章指标发送给软件，软件负责入库、评分、筛选、预览、种子池管理和导出。

## 主界面

主界面分为四块：

1. 仪表盘：显示种子数量、文章数量和当前最高分。
2. 文章表格：显示标题、账号、分类、阅读、点赞、评论、频率、评分和链接。
3. 控制中心：包含筛选条件、种子池、微信运行配置和日志。
4. 顶部菜单：提供加载示例、导出、加载插件、预览、标星账号、本地桥冒烟测试等操作。

## 第一次使用

1. 启动软件。
2. 打开左侧控制中心里的 WeChat 页面。
3. 确认 SQLite 数据库路径，建议放到稳定磁盘目录。
4. 确认插件目录，默认是程序旁边的 `plugins`。
5. 除非已经准备好手机自动化方案，否则先不要启用 ADB 自动化。
6. 点击保存运行配置。
7. 在 Plugins 菜单里点击加载插件。
8. 在 Actions 菜单里点击发送本地桥冒烟载荷，确认链路可用。

## 真实数据接入

真实接入时，需要一个本地适配器把文章指标发送到 `127.0.0.1:<桥端口>`，默认端口是 `9000`。软件只接受两个端点：

- `/mp/getappmsgext`
- `/mp/appmsg_comment`

文章指标载荷示例：

```json
{
  "endpoint": "/mp/getappmsgext",
  "title": "文章标题",
  "url": "https://example.local/article",
  "account_name": "公众号名称",
  "gzh_id": "公众号ID",
  "category": "科技",
  "article_count_30d": 12,
  "appmsgstat": {
    "read_num": 24000,
    "like_num": 1200,
    "old_like_num": 300
  }
}
```

评论载荷示例：

```json
{
  "path": "/mp/appmsg_comment?action=getcomment",
  "title": "文章标题",
  "url": "https://example.local/article",
  "comment_count": 88
}
```

未知端点会被忽略，这是为了缩小数据入口边界，避免把无关流量写入数据库。

## 筛选和评分

评分公式：

```text
Score = W_eng * EngagementRate + W_comm * CommentDensity + W_freq * FrequencyScore
EngagementRate = (like_num + old_like_num) / read_num
CommentDensity = comment_num / read_num * 10000
FrequencyScore = min(100, article_count_30d / 30 * 100)
```

你可以在筛选面板调整行业、关键词、权重和最低阅读量。评分越高，代表文章互动密度、评论密度和账号活跃度综合表现越好。

## 种子池

种子池用于保存重点公众号。支持：

- 新增或更新公众号。
- 删除选中公众号。
- 导出种子 CSV。
- 选中文章后按 Ctrl+S，把文章所属公众号保存为种子。

## 导出

File 菜单支持三种导出：

- 文章 CSV。
- 文章 JSON。
- 种子 CSV。

导出路径由用户手动选择，软件不会自动上传文件。

## 快捷键

- 空格：预览选中文章。
- Ctrl+S：把选中文章的账号加入种子池。
- Esc：重置筛选条件。


## 中英文切换和工具提示

软件在帮助菜单里提供运行时语言切换。点击“帮助 -> English”可以切到英文界面；点击“Help -> 中文”可以切回中文界面。当前语言会保存到本地运行配置文件，下次启动时自动恢复。

软件界面保持中英文分离。中文模式显示中文菜单、标签、表头、占位提示、工具提示和内置说明书；英文模式显示英文菜单、标签、表头、占位提示、工具提示和内置说明书。正常使用时不应该出现中英文混在同一个控件里的情况。

主要操作控件都带有工具提示。鼠标悬停在筛选控件、种子输入框、微信运行配置、导出操作、预览操作和本地桥冒烟操作上，可以看到这个控件干什么、怎么用，以及对应安全边界。

## 软件内置使用说明书

左侧控制中心新增“说明书”页，里面放置软件内置使用说明，方便不打开外部文档也能完成基本操作：

1. 配置微信接入页。
2. 保存配置并加载插件。
3. 运行本地桥冒烟测试。
4. 选择 A、B、C 其中一种接入方案。
5. 调整筛选条件和评分权重。
6. 预览选中文章并把公众号收藏为种子。
7. 维护种子池。
8. 导出文章或种子。
9. 遵守本地隐私和凭证边界。

关于窗口也会显示当前语言版本的使用说明。
