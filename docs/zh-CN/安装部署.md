# 安装部署

## Linux 使用发布包

```bash
tar -xzf premium-content-radar-*-linux-*.tar.gz
cd premium-content-radar-*-linux-*
./bin/premium-content-radar
```

如果插件没有被发现，请确认运行目录中 `plugins` 目录仍然和程序在同一个发布包结构里。

## Linux 从源码构建

Ubuntu 或 Debian 类系统依赖：

```bash
sudo apt-get update
sudo apt-get install -y cmake g++ qt6-base-dev qt6-tools-dev libqt6sql6-sqlite libgl1-mesa-dev
```

构建和测试：

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j2
ctest --test-dir build --output-on-failure
QT_QPA_PLATFORM=offscreen ./build/premium-content-radar --bridge-smoke
./build/premium-content-radar
```

## Windows 从源码打包

需要安装：

- Qt 6。
- CMake。
- 对应的 C++ 编译器。
- PowerShell。

执行：

```powershell
$env:VERSION="1.0.1"
.\scripts\package-windows.ps1
```

## Docker

```bash
docker buildx build --platform linux/amd64,linux/arm64 -t premium-content-radar:latest .
```

Docker 主要用于构建、验证和远程桌面实验。生产使用更推荐原生桌面包。

## 卸载

删除解压后的软件目录即可。可选清理项：

- SQLite 数据库：路径在 WeChat 配置页里设置。
- 运行配置：系统用户配置目录下的 `premium-content-radar/settings.json`。
- 用户导出的 CSV 或 JSON 文件。
